This is a custom shop page using the same hand-drawn site style. Use the filter buttons to browse by product type. The sample catalogue includes sample products across **stickers**, **prints**, **keychains / pins** and **apparel**.

Use the **Products** collection in Pages CMS to add as many images as each product needs. The first image appears on the shop grid. The popup uses left/right arrows to move through the rest.
